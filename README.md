# Json Path Translate extension
This is a little extension that want to solve a little problem to my working flow, angular translations files usually works with *.json ex: en.json, this files tend to grow with a certain facility, so search and get the path to use a simple translation it's a little anoying so there's the solution in form of extension.

## Features

Search a word in a selected json file and return the path with separation that contains the value of the word
ex: **grid.card.hero.form.title**


**Enjoy!**
